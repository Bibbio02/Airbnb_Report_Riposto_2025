# Airbnb_Report_Riposto_2025
Analisi Data-Driven del turismo a Riposto: integrazione dati ISTAT 2024 e Airbnb 2025 sviluppata interamente in Python. Benchmarking territoriale con Giardini Naxos e Zafferana Etnea tramite web scraping e data cleaning avanzato. Conclude con una SWOT Analysis per supportare strategie amministrative basate sull'evidenza dei dati.
